# Andres Real Estate Developer Website

A professional website showcasing Andres, a real estate developer and tennis enthusiast. The site highlights his portfolio of properties, professional background, and passion for real estate development.

## Features

- **Responsive Design**: Mobile-friendly layout that works on all devices
- **Hero Section**: Eye-catching landing section with call-to-action
- **Properties Portfolio**: Showcase of featured real estate listings with pricing and details
- **About Section**: Professional background and personal interests (including tennis)
- **Contact Form**: Easy-to-use contact form for inquiries
- **Smooth Navigation**: Sticky navigation bar with smooth scrolling
- **Interactive Elements**: Hover effects, animations, and form feedback

## Project Structure

```
├── index.html       # Main HTML file with page structure
├── styles.css       # Responsive styling and layouts
├── script.js        # Interactive functionality and animations
└── README.md        # This file
```

## Getting Started

Simply open `index.html` in your web browser to view the website. No server or build tools required.

### Features Included

1. **Navigation Bar** - Sticky navigation with links to all sections
2. **Hero Section** - Professional branding with featured content
3. **Properties Grid** - Four featured properties with details and call-to-action buttons
4. **About Section** - Professional background and tennis accomplishments
5. **Contact Section** - Contact form and business information
6. **Responsive Layout** - Adapts to mobile, tablet, and desktop screens

## Technologies Used

- **HTML5** - Semantic markup
- **CSS3** - Flexbox, Grid, and animations
- **JavaScript** - Form handling and interactive features

## Customization

To update content, simply edit the HTML file:
- Change property listings in the `.properties-grid` section
- Update contact information in the `.contact-info` section
- Modify professional details in the `.about-content` section

## Styling

The website uses a modern color scheme:
- Primary Color: #2c3e50 (Dark blue)
- Secondary Color: #3498db (Bright blue)
- Accent Color: #e74c3c (Red)

All styles are in `styles.css` and use CSS variables for easy customization.

## Browser Support

The website works on all modern browsers:
- Chrome/Chromium
- Firefox
- Safari
- Edge
